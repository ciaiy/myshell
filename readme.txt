程序目标:
    单个命令:   ls
    带-l到多个参数的命令:   ls -l /temp
    带输出重定向的命令: ls -l > a
    带输入重定向的命令: wc -c < a
    带管道命令: ls -l | wc -c
    后台运行符可以添加到命令的最后面:   ls &
